# A/B-тест маркетинговых кампаний

## Описание проекта

В проекте проведён анализ A/B-теста двух рекламных кампаний (Control и Test).

Цель — определить, какая из кампаний эффективнее с точки зрения привлечения пользователей, вовлечённости и конверсии, а также сформировать рекомендации по выбору стратегии.

В работе сделан акцент не только на проверке статистической значимости, но и на корректности расчёта метрик и интерпретации результатов с точки зрения бизнеса.

---

##  Гипотезы

- **H₀:** Различий между кампаниями по ключевым метрикам нет  
- **H₁:** Между кампаниями есть статистически значимые различия  

---

##  Данные

Данные по двум рекламным кампаниям:

- Control Campaign  
- Test Campaign  

Каждая строка — один день кампании.

Содержат:

- расходы на рекламу  
- показы (impressions)  
- клики  
- поисковые действия  
- просмотры контента  
- добавления в корзину  
- покупки  
 

---

##  Подготовка данных

- Приведены названия колонок к единому формату  
- Проверены пропуски  
- Удалена строка с некорректными данными  
- Проверена корректность данных  

Дополнительно рассчитаны метрики на уровне отдельных наблюдений:

- CTR  
- Conversion Rate  
- CPA  
- Cost per Conversion  

---

##  Анализ

- Рассчитаны средние значения метрик в контрольной и тестовой группах  
- Проведено A/B-тестирование с использованием Welch t-test  

Рассчитаны:

- p-value  
- effect size  

Сравнены ключевые этапы воронки:

- клики  
- интерес  
- добавление в корзину  
- покупки  

---

##  Результаты

### Статистически значимые различия:

- CTR выше в Test Campaign (**p = 0.0003**)  
- Added to Cart выше в Control Campaign (**p = 0.0001**)  

### Без значимых различий:

- Purchases  
- Conversion Rate  
- CPA  
- Cost per Conversion  
- Website Clicks  
- Searches  
- Content Viewed  

---

##  Выводы и рекомендации

Test Campaign лучше привлекает пользователей (выше CTR).

Control Campaign показывает лучший результат на этапе добавления в корзину.

При этом различий в ключевых бизнес-метриках (покупки и стоимость привлечения) не обнаружено.

### Рекомендация:

Использовать **Test Campaign**, так как она:

- эффективнее генерирует трафик  
- не ухудшает конверсию  
- не увеличивает стоимость привлечения
